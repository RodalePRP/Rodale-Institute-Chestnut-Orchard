GitHub Pages setup

These are the minimal steps needed to publish this file as a website.

1

Create a repository

Use a repository named yourusername.github.io for a user site, or any repo name for a project site.

2

Add this file as index.html

Commit it to the repository root, or place it in a docs/ folder if you prefer that Pages source.

3

Enable GitHub Pages

In repository settings, select the branch and folder to publish, then save.

4

Customize content

Edit text, colors, and links. Add sections for publications, contact info, or downloads.

Suggested file structure

your-repo/ ├── index.html ├── assets/ │ ├── images/ │ └── files/ └── README.md

Tip

If this is a project site instead of a user site, your URL will usually be https://yourusername.github.io/your-repo/.
